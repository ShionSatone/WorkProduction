// ゲームの説明
//==================================================================
 ネコ1匹とネズミ3匹に分かれてそれぞれの目的を達成しよう！
 以下それぞれの役割↓
 
 [ネコ]
 ネズミ3匹を駆除しよう！
 ギミックやアイテムを駆使して全員駆除すればクリアです。
 1匹でも生き残るとネコの負けなので、頑張って倒しましょう。
 
 [ネズミ]
 時間いっぱい1匹のネコから逃げきろう！
 ギミックを駆使したり、やられてしまった味方を助けて生き残りましょう。
 1匹でも生き残れば勝ちです。逃げるもよし、味方と協力するもよし。
 
 

※このゲームは「４人プレイのみ」対応しています。
  双方のルールやギミック、アイテムについての具体的な説明は、
  「説明.pdf」をご覧ください。
  見ておくと場を有利に進められるかも…？

// アピールポイント
//******************************************************************
・死亡時の演出
 マルチゲームではいかにわかりやすいかが大切だと考え、その中でも私が担当したプレイヤーの死亡時の演出は、
 特にユーザーのことを考え制作しました。
 制作当時、広大なマップなのでどうしても味方が死亡した際にその味方がどこで死んだのかがわからず、
 回復しに行けなくなることが問題でした。
 そこで死亡したプレイヤーの上に大きく「HELP!!」と表示したり、死亡したプレイヤーへの矢印を設置することによって
 味方が助けやすくなるようになりました。
 矢印は死亡したプレイヤーの数出すことができるので、何人死亡しているのかも把握しやすくなりました。
 復活中のゲージも制作し、より回復してる様子がわかるようにもしました。
