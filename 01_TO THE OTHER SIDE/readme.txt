//==================================================================
// ゲームの説明
//==================================================================
 プレイヤーの女の子と影を切り替えてボスを倒していくゲームです。
 影をうまく使いこなして仕掛けを作動し、道を切り開いていきましょう。
 雫を全て集めると…？
 
//==================================================================
// ゲームの操作方法
//==================================================================
※操作：キーボード

・移動：A,D
・ジャンプ：SPACE
・影切り替え：ENTER

//******************************************************************
// アピールポイント
//******************************************************************
・影と人を切り替えて進む
 ステージを行き来するパズル要素が欲しかったので、影を使用したギミックを実装しました。
 1つのステージでも影でしか進めない道を用意することで、スイッチをオンにすることで
 道が現れる達成感が出るようにしました。
 これにより、一見情報量も少なくすぐに進めそうなステージも、謎を解くことで長い時間遊べるように
 なりました。
 
・ボス戦のBGMの作曲
 ボス戦のBGMは私がゲームの雰囲気に合わせて作曲したものです。
 ただフリー素材を使用するでも良かったのですが、どの素材も私が思い描く、
 戦闘のイメージにあったBGMではありませんでした。
 そこで趣味程度に作曲していたこともあり、せっかくならと自身の思い描くイメージを作ってみました。
 結果的に物凄く雰囲気のある戦闘になり、作曲してよかったと感じました。
